# Lung Nodule Segmentation (U-Net)

## 📌 Project Description
This project performs automatic lung nodule segmentation on CT images using a U-Net based deep learning model.

## 🧠 Model
- Architecture: U-Net
- Framework: PyTorch
- Input size: 256x256

## 📊 Results
- Dice Score: 0.9391
- IoU: 0.8854
- Recall: 1.0000

## 📁 Files
- report.pdf → detailed report
- notebook.ipynb → training code
- images/ → results and graphs

## 🚀 How to Run
Open notebook.ipynb in Google Colab and run all cells.
